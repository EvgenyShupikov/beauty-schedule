# Описание

Тестовое задание "Расписание салона красоты".

## Доступные скрипты

`npm start` - запуск
`npm test` - тестирование

## Технологии проекта

Проект сделан на основе [Create React App](https://github.com/facebook/create-react-app):
- React
- TypeScript
- Jest

Дополнительно установлены:
- Bootstrap
- FontAwesome
- Redux-Saga
- Faker
